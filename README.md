# Solar System Game

An interactive Solar System experience built with HTML, CSS, and JavaScript. Explore the planets, learn interesting facts, and test your knowledge through quizzes and challenges.

Live Demo: https://solar-systemgame.vercel.app/


## Features

### Planet Exploration

* Explore all 8 planets in the Solar System.
* Read facts and information about each planet.
* Learn about their unique characteristics.

### Quiz Mode

* Test your knowledge with space-related questions.
* Challenge yourself and improve your score.

### Interactive Tutorial

* Includes a tutorial to help users understand how everything works.
* Easy to follow and beginner-friendly.

### Animations and Effects

* Smooth animations and transitions.
* Designed to make learning about space more engaging.

### Browser-Based

* No downloads or installations required.
* Works directly in your browser.

## How to Use

1. Open the website.

2. Browse through the planets and read about them.

3. Follow the tutorial if you're new to the website.

4. Try the quiz mode and see how much you know about the Solar System.

5. Keep exploring and learning.

## Installation

Clone the repository:

```bash
git clone https://github.com/Rishi-Rahul/Solar-Systemgame.git
```

Open the project folder and launch `index.html`.

No extra setup is required.
